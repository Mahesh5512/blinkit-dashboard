# blinkit-dashboard
This project is an interactive sales dashboard created using Blinkit sales data to understand how different products and outlets are performing.
The main goal of this dashboard is to analyze sales, customer ratings, outlet types, and product categories in a clear and visual way so that anyone can easily understand the business performance.

-What this dashboard shows

Total sales, average sales, average rating, and number of items

Sales breakdown by item type (fruits, snacks, dairy, household, etc.)

Comparison of Low Fat vs Regular products

Sales analysis by outlet size (Small, Medium, High)

Sales by outlet location (Tier 1, Tier 2, Tier 3)

Outlet establishment trend over the years

Performance comparison between Grocery Stores and Supermarkets

- Why I built this project

I built this dashboard to:

Practice data analytics and visualization

Understand how retail sales data can be analyzed

Improve my skills in creating interactive business dashboards

Show how insights can help in better decision-making

- Tools Used

Power BI

Data Cleaning & Data Modeling

Interactive charts and filters

- Key Insights

Medium-sized outlets contribute the highest sales

Tier 3 locations perform better compared to Tier 1 and Tier 2

Regular fat products generate more sales than low-fat ones

Supermarkets have higher sales compared to grocery stores
